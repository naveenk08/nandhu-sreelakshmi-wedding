

# Wedding Invitation Website

## Overview
A stunning, mobile-first single-page wedding invitation with elegant animations, celebratory effects, and a warm color palette (gold, cream, pastel pink, maroon).

## Design
- **Fonts**: Great Vibes (cursive names), Playfair Display (headings), Poppins (body)
- **Colors**: Gold (#D4AF37), Cream (#FFF8E7), Pastel Pink (#F8C8DC), Royal Maroon (#800020), White
- **Style**: Soft shadows, rounded corners, generous spacing, elegant and festive

## Sections (top to bottom)

### 1. Hero Landing
- Fullscreen with soft gradient background + floating flower/sparkle particles
- Bride & Groom names in large cursive, tagline below
- Animated "Open Invitation" button that scrolls down
- Confetti burst on page load

### 2. Countdown Timer
- Days, hours, minutes, seconds with animated ticking
- Elegant card-style display

### 3. Couple Showcase Carousel
- Horizontal auto-play image carousel with placeholder couple photos
- Fade/zoom transitions, pause on touch

### 4. Invitation Message
- Formal wedding card styled section
- Elegant centered typography, fade-in on scroll

### 5. Event Details
- Card layout with icons for date, time, venue, address
- Subtle hover animations

### 6. Location
- Embedded Google Maps iframe
- "Open in Maps" button
- Rounded card with shadow

### 7. Photo Gallery
- Masonry/grid layout with placeholder images
- Lightbox modal on click to enlarge
- Scroll-triggered animations

### 8. RSVP / Contact
- RSVP button, WhatsApp message button
- Contact info display

### 9. Footer
- Share button, Save-the-Date .ics download
- Background music toggle (mute/unmute)

## Technical Approach
- Intersection Observer for scroll-triggered animations
- CSS animations + Framer Motion for transitions
- Lazy loading for images and sections
- Fully responsive, mobile-first
- Reusable components for cards, sections, animations
